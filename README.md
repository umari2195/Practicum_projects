# Онлайн-игры «Секреты Темнолесья»
#### Использованные инструменты - 'DBeaver'
Команда игры планирует привлекать платящих игроков и продвигать покупку эпических предметов с помощью рекламы. 
- Мы выяснили, какая доля игроков покупает внутриигровую валюту и есть ли зависимость доли платящих игроков от расы персонажа;
- детально изучили, как происходит покупка эпических предметов внутри игры.
